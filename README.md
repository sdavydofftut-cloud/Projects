# Projects — Проекты автоматизированного тестирования

## 🛠️ Инструменты и технологии

- **Язык и подходы:** Python (ООП, обработка исключений, параметризация, фикстуры, моки)
- **Тестирование:** Pytest, Unit‑тесты, Selenium, Requests, Allure, Page Object Model
- **Веб и API:** REST API, HTTP, JSON
- **Инфраструктура и IDE:** Git, VS Code

---

## 🍔 Автоматизация тестирования сервиса заказа бургеров Stellar Burgers

### Задача 1. Проверка функциональности класса `Burger`

**Цель:** обеспечить надёжную проверку функций класса `Burger` в программе для заказа бургеров.

**Решение:**
- Разработан набор Unit‑тестов со 100 % покрытием для класса `Burger`.
- Проект организован в виде двух пакетов: отдельный пакет для кода программы и отдельный — для тестов.

**Результат:**
- Гарантирована корректность работы всех методов класса.
- Снижен риск появления ошибок при дальнейшей разработке.

**Папка:** `Unit-tests`

---

### Задача 2. Автоматизация проверки API сервиса заказа бургеров

**Цель:** автоматизировать проверку API сервиса для сокращения времени регрессионного тестирования.

**Решение:**
- Реализованы автоматизированные тесты API на Python 3.14 с использованием Pytest и библиотеки `requests`.
- Настроена генерация отчётов через Allure‑Pytest.
- Структура позволяет легко масштабировать набор тестов и переиспользовать данные/клиенты между сценариями.

**Результат:**
- Сокращено время регрессионного тестирования.
- Обеспечено быстрое выявление ошибок в API.

**Папка:** `API-tests`

---

### Задача 3. Проверка основной функциональности сервиса в реальных браузерах

**Цель:** проверить основную функциональность сервиса Stellar Burgers в реальных браузерах.

**Решение:**
- Созданы UI‑автотесты для веб‑приложения.
- Реализована поддержка кросс‑браузерного тестирования (Chrome, Firefox).
- Спроектирована модульная структура проекта по принципам Page Object Model (POM).

**Результат:**
- Обеспечена стабильная работа пользовательского интерфейса в целевых браузерах.
- Ускорен процесс проверки основной функциональности.

**Папка:** `UI-tests`
