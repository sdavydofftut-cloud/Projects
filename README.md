**Projects**  
**Проекты автоматизированного тестирования**

**Инструменты и технологии:** Python (ООП, обработка исключений, параметризация, фикстуры, моки), Pytest, Unit‑тесты, Selenium, Allure, Page Object Model, Git, VS Code, REST API, HTTP, JSON.

**Автоматизация тестирования сервиса заказа бургеров Stellar Burgers**

**Задача 1.** Обеспечить надёжную проверку функциональности класса Burger в программе для заказа бургеров.  
**Решение.** Разработал набор Unit‑тестов с 100 % покрытием для класса Burger, организовал проект в виде двух пакетов: код программы и тесты.  
**Результат.** Гарантирована корректность работы всех методов класса, снижен риск ошибок при дальнейшей разработке.  
**Папка: Unit-tests**

**Задача 2.** Автоматизировать проверку API сервиса заказа бургеров для сокращения времени регрессионного тестирования.  
**Решение.** Реализовал автоматизированные тесты API на Python 3.14 с использованием Pytest и библиотеки requests, настроил генерацию отчётов через Allure‑Pytest, спроектировал модульную структуру проекта по принципам POM.  
**Результат.** Сокращено время регрессионного тестирования, обеспечено быстрое выявление ошибок в API.  
**Папка: API-tests**

**Задача 3.** Проверить основную функциональность сервиса Stellar Burgers в реальных браузерах.  
**Решение.** Создал UI‑автотесты для веб‑приложения с поддержкой кросс‑браузерного тестирования (Chrome, Firefox).  
**Результат.** Обеспечена стабильная работа пользовательского интерфейса в целевых браузерах, ускорен процесс проверки основной функциональности.  
**Папка: UI-tests**
