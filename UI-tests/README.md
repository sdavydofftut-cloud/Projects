# UI-tests: Автотесты для UI приложения Stellar Burgers

## 📋 1. Общее описание проекта

Проект представляет собой набор автоматизированных UI‑тестов, предназначенных для проверки основной функциональности веб‑приложения Stellar Burgers. Тесты реализуют ключевые пользовательские сценарии и обеспечивают контроль качества интерфейса и бизнес‑логики.

---

## 🛠️ Технологии и инструменты

- **Язык программирования:** Python 3.
- **Фреймворк для тестирования:** Pytest.
- **Инструмент автоматизации:** Selenium WebDriver.
- **Паттерн проектирования:** Page Object (для структурирования кода, удобства поддержки и переиспользования).
- **Инструмент для отчётов:** Allure (для генерации наглядных отчётов о прохождении тестов).

---

## **✅ 3. Проверенная функциональность**
**3.1. Основная функциональность**
**Навигация:**
Проверен переход на страницу «Конструктор» по клику на соответствующий элемент меню.

Проверен переход на страницу «Лента заказов».

Взаимодействие с ингредиентами:

При клике на ингредиент появляется всплывающее окно с его деталями.

Всплывающее окно успешно закрывается при клике на кнопку‑крестик (X).

Добавление в заказ:

Реализована проверка увеличения счётчика ингредиента при его добавлении в заказ (тест проходит в Chrome и Firefox).

3.2. Раздел «Лента заказов»
Статистика заказов:

При создании нового заказа проверено увеличение счётчика «Выполнено за всё время».

Проверено увеличение счётчика «Выполнено за сегодня».

Отслеживание статуса:

После оформления заказа его номер успешно отображается в разделе «В работе».

🚀 4. Запуск тестов
Для выполнения тестов необходимо иметь установленный Python и зависимости из файла requirements.txt.

4.1. Установка зависимостей
bash
pip install -r requirements.txt
4.2. Запуск тестов
По умолчанию тесты запускаются в браузере, указанном в фикстуре driver. Фикстура параметризована на два браузера: Chrome и Mozilla Firefox.

Базовый запуск:

bash
pytest -v
Запуск с генерацией Allure‑отчёта:

Запустите тесты с сохранением результатов в папку allure-results:

bash
pytest --alluredir=allure-results
Сгенерируйте HTML‑отчёт из полученных данных:

bash
allure serve allure-results
